Lesson 0-1: compilation and execution

First, we learn a basic procedure to run Fortran programs on a computer. To run a program, one often follow the following procedure: (i) write a Fortran code in .f90 format. (ii) Compile the fortran code and create an executable file. (iii) Run the execurable file to obtain results.

To learn this basic procedure, we try to run "Hello world" program, which is a traditional program to learn the programing language (see e.g. Wikipedia). As the first step, we write the following Fortran code with your favolit editor and save it as lesson0_1.f90.

(lesson0_1.f90) 
program main
	    
  write(*,*)"Hello world!!"

end program main

The first line program main indicates the start of the program block and it ends by end program main. Here, main is the name of the program, and this name can be any name. For simplicity, here we use main. The second line write(*,*)"Hello world!!" means that the program writes "Hello world!!". The write sentence is often used to output results of programs, as will be shown later.n

Once the above Fortran code (lesson0_1.f90) is prepared, we then compile the code to provide an executable file. You can compile the program by typing the following commands on your terminal:

gfortran lesson0_1.f90 -o hello

Here, we used gfortran compiler, which is one of the most popular Fortran compiler, but you can use your favorite compiler instead. By typing the above commnds, the Fortran source file (lesson0_1.f90) is compiled, and the executable file (hello) is generated. Here, the name of the executable file is defined with -o xoption. You can change the names of the source code and the executable file as you like.
